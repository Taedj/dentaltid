  1. Open your Firebase Console (https://console.firebase.google.com/).
   2. Go to Firestore Database.
   3. Navigate to the collection: `users` -> `[User_UID]` -> `profile` -> `info`.
   4. Edit the following fields (Case-Sensitive):
       * `plan`: Change the value to `SubscriptionPlan.enterprise` (must be the full string).
       * `isPremium`: Change the value to `1` (as a Number).
       * `status`: Ensure it is `SubscriptionStatus.active`.
   5. Click Update.
   6. Restart the Windows App.
